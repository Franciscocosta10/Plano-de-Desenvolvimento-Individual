# Plano-de-Desenvolvimento-Individual

Disciplina: Algoritmos Avançados
Curso: Engenharia de Software

1. Objetivo do Semestre

Minha motivação na disciplina de Algoritmos Avançados é aprofundar meu conhecimento em lógica e resolução de problemas computacionais, entendendo como construir soluções mais eficientes e bem estruturadas. 

Ao longo do semestre, desejo dominar melhor conceitos como análise de complexidade, recursão, programação dinâmica, estruturas de dados e algoritmos aplicados a grafos, desenvolvendo a capacidade de analisar problemas e escolher a melhor abordagem para resolvê-los. 

Esse aprendizado é importante para minha formação em Engenharia de Software, pois algoritmos são fundamentais para o desenvolvimento de sistemas eficientes e escaláveis, além de contribuir para minha evolução profissional na área de tecnologia e desenvolvimento de software.
